Dynamic Forum

Ajax, PHP, and Data-Visualization

Utilizing jQuery's AJAX API and PHP I created a simple forum for the video game community League of Legends. In this forum users can post and delete information on characters from the game. These posts are done through AJAX requests to an external database utilizing PHP and MySQL. Further functionality includes sorting data by name, recently posted, or by the most popular subject. Data visualization was created utilizing chart.js to dynamically pull in current and updated data from the database in real-time to display what characters are currently present in the data set.
